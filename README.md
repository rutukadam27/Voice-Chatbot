# 🎙️ Voice-Chatbot

## 📌 Description
Voice-Based Chatbot is a Python application that interacts with users using voice commands. It listens to user input through a microphone, processes the query, and responds using text-to-speech.

---

## 🚀 Features
- Voice input using microphone
- Speech recognition
- Text-to-speech response
- Basic chatbot interaction logic
- Hands-free user experience

---

## 🛠️ Technologies Used
- Python
- SpeechRecognition
- PyAudio
- pyttsx3

---

## 📂 Project Structure
Voice-Chatbot/
├── main.py        # Main chatbot logic
├── requirements.txt
└── assets/        # (optional: audio/screenshots)

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/rutukadam27/Voice-Chatbot-Python.git

2. Install required libraries:
   pip install SpeechRecognition pyttsx3 PyAudio

3. Run the program:
   python main.py

---

## ⚠️ Requirements
- Microphone access
- Python 3.x installed
